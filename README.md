## 中文

就和名称一样: 将`.pptx`一键式的打包为`.png`图片.  
提供一个Windows桌面端`.exe`应用和一个可调用的Python库.  

- [EXE应用](https://github.com/Water-Run/pptx2img/releases/tag/pptx2img), 下载后执行`pptx2img.exe`即可  
- [Python库](./lib/README.md), `pip install pptx2img`, 详见文档  

开源于[GitHub](https://github.com/Water-Run/pptx2img), 遵循`GPL`协议.  

## English

As the name suggests: easily convert `.pptx` files into `.png` images with a single click.  
It provides both a Windows desktop `.exe` application and a Python library that can be integrated into your own projects.

- [EXE Application](https://github.com/Water-Run/pptx2img/releases/tag/pptx2img): download and run `pptx2img.exe` to get started  
- [Python Library](./lib/README.md): install via `pip install pptx2img`, see documentation for usage examples  

Open-sourced on [GitHub](https://github.com/Water-Run/pptx2img) under the `GPL` license.
